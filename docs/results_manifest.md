# Results Manifest

The main entry point is:

```text
notebooks/01_reproduce_report_results.ipynb
```

By default, the notebook loads and displays saved report-ready outputs, runs the quick Heston COS/Carr--Madan validation, and reports missing files with regeneration routes. Long neural-network training runs are disabled by default.

In Google Colab, the notebook can mirror every generated CSV, text file, JSON file, figure, progress log, and final zip to:

```text
MyDrive/MFE_neural_hedging_report_outputs/
```

Set `RUN_MODE = "full_generation"` in the notebook to generate the clean output set through `src/report_generation.py`.

The generated `results/report/report_output_manifest.csv` includes both local and Drive paths with columns:

```text
experiment, mode, output_type, local_path, drive_path, used_in_report,
report_table_or_figure, timestamp, generated_in_current_run, notes
```
